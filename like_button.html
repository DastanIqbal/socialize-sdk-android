Socialize Like Button¶

In v1.4 of the Socialize SDK for Android is a new stand-alone Like Button. This is simply the “like” capablity of Socialize contained in a self-standing UI element which can be easily customized to suit your app.

Displaying the Like Button¶

The Like Button can either be included via standard Android XML Layout files, or programmatically in your Activity code.

Using XML layout¶


<?xml version="1.0" encoding="utf-8"?>
<LinearLayout
        xmlns:android="http://schemas.android.com/apk/res/android"
        android:orientation="vertical"
        android:layout_width="fill_parent"
        android:layout_height="fill_parent">

        <com.socialize.ui.actionbutton.SocializeLikeButton
                xmlns:socialize="http://getsocialize.com"
                android:id="@+id/socializeLikeButton"
                android:layout_width="150dp"
                android:layout_height="45px"
                android:background="#555555"
                android:textColor="#ffffff"
                socialize:entity_key="your_entity_key"
                socialize:entity_name="your_entity_name"
                socialize:src_active="@drawable/like_image_on"
                socialize:src_inactive="@drawable/like_image_off"
                socialize:src_disabled="@drawable/like_image_disabled"
                socialize:text_active="Unlike"
                socialize:text_inactive="Like"
                socialize:share_location="true"
                socialize:auto_auth="true"
                socialize:show_count="true"
                socialize:share_to="facebook"/>

</LinearLayout>

If you don’t want to (or can’t) set the entity key as a fixed value in XML, you can set it at runtime in code:

        // Retrieve a reference to your like button based on it's ID
        SocializeLikeButton button = (SocializeLikeButton) findViewById(R.id.socializeLikeButton);
        button.setEntity(Entity.newInstance("foo", "bar"));

Using programmatic configuration¶

If you prefer to do things the old-fashioned way, you can set all the parameters programmatically.

import android.app.Activity;
import android.os.Bundle;
import android.view.ViewGroup;

import com.socialize.entity.Entity;
import com.socialize.networks.SocialNetwork;
import com.socialize.ui.actionbutton.ActionButtonConfig;
import com.socialize.ui.actionbutton.SocializeLikeButton;

public class SampleActivity extends Activity {

        @Override
        protected void onCreate(Bundle savedInstanceState) {
                super.onCreate(savedInstanceState);

                // Set your content view as you would do normally
                setContentView(R.layout.empty);

                // Create a like button
                SocializeLikeButton button = new SocializeLikeButton(this);

                // Set the entity on the button
                button.setEntity(Entity.newInstance("foo", "bar"));

                // Get the config object to set the configuration options
                ActionButtonConfig config = button.getConfig();
                config.setBackgroundColor(Color.parseColor("#555555"));
                config.setImageResIdOn(R.drawable.action_button_on);
                config.setImageResIdOff(R.drawable.action_button_off);
                config.setImageResIdDisabled(R.drawable.action_button_disabled);
                config.setShareToNetworks(SocialNetwork.FACEBOOK);
                config.setAutoAuth(true);
                config.setTextColor(Color.WHITE);
                config.setShowCount(true);
                config.setTextOff("Like");
                config.setTextOn("Unlike");

                // We are going to add this to our main view, but it can be added to any view group
                ViewGroup main = (ViewGroup) findViewById(R.id.main);
                main.addView(button);
        }
}
